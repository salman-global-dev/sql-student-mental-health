# 📊 Analyzing Students' Mental Health (SQL Analysis)

## 📌 Project Overview
This project explores a dataset from a Japanese international university to analyze the mental health of 268 students. The goal was to determine if the **Length of Stay** impacts diagnostic scores for depression (**PHQ-9**), social connectedness (**SCS**), and acculturative stress (**ASISS**).

## 🛠️ Tech Stack
* **Database:** PostgreSQL
* **Tools:** SQL, DataCamp Datalab
* **Techniques:** Data Aggregation, Filtering, Rounding, and Trend Analysis

## 🔍 Key Findings
Using SQL queries to aggregate scores by length of stay, I discovered:
* **Depression Risk:** Average PHQ-9 scores peak at **13.0** for students staying **10+ years**.
* **Social Connection:** The Social Connectedness Scale (SCS) showed a downward trend the longer a student stayed, dropping to an average of **32.0**.
* **Adaptation Paradox:** While depression scores rose, Acculturative Stress (ASISS) decreased for the longest-staying students, suggesting cultural adaptation but lower personal well-being.
